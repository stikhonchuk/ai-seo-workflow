## Buying Guide Content Prompt

### Template Prompt
```
Write a comprehensive buying guide article in Russian for [YOUR-DOMAIN] with the following specifications:

PRIMARY KEYWORD: [insert primary keyword]
SECONDARY KEYWORDS: [insert 3-5 secondary keywords]
TARGET WORD COUNT: 2,000-2,500 words
TARGET AUDIENCE: [specify audience - e.g., Russian women aged 25-45 looking for comfortable work shoes]

ARTICLE STRUCTURE:
1. Catchy title with primary keyword
2. Introduction (150-200 words) - Hook reader with problem/benefit
3. Quick comparison table of top 5-7 products
4. "How to Choose" section with key buying factors
5. Detailed reviews of top products (150-200 words each)
   - Include pros, cons, best for, price range
6. Comparison section (comparing top 2-3 options)
7. FAQ section (5-8 questions with answers)
8. Conclusion and final recommendation

SEO REQUIREMENTS:
- Primary keyword in: title, first paragraph, one H2, conclusion
- Secondary keywords distributed naturally
- 3-5 internal links to [YOUR-DOMAIN] products/categories
- 1-2 external links to authoritative sources
- Meta title (50-60 characters)
- Meta description (150-160 characters)

TONE: Professional, helpful, authoritative but approachable
LANGUAGE: Russian
MARKET: Russian e-commerce customers

Include specific product recommendations with:
- Product names and brief descriptions
- Key features and benefits
- Price range in rubles
- Who it's best suited for
```

### Example Usage
```
Write a comprehensive buying guide article in Russian for [YOUR-DOMAIN] with the following specifications:

PRIMARY KEYWORD: лучшие кроссовки для бега
SECONDARY KEYWORDS: беговые кроссовки женские, как выбрать кроссовки для бега, кроссовки для плоскостопия, кроссовки для бега 2026
TARGET WORD COUNT: 2,200 words
TARGET AUDIENCE: Russian women aged 25-45 interested in running and fitness

[Follow template structure above]
```

```

- 1-2 external authoritative sources

TONE: Instructional, friendly, encouraging
LANGUAGE: Russian
FORMAT: Clear, scannable with short paragraphs

Include:
- Specific, actionable steps
- Expected results
- Time estimates
- Difficulty level
```

```
LANGUAGE: Russian
```

```
- Are specific and actionable

```

```
PRIMARY KEYWORD: [insert keyword]
NUMBER OF ITEMS: 7-15 (odd numbers work well)

1. Numbered title: "[Number] [Topic] в [Year]"
2. Introduction (150-200 words)
3. Quick summary list (numbered list of all items)
4. Detailed entries for each item (150-250 words each):
   - Item name as H3
   - Why it made the list
   - Key features/reasons
   - Who it's for or when to use
5. Summary/comparison of top 3
6. FAQ section
7. Conclusion

SEO REQUIREMENTS:
- Number in title and H1
- Include year for freshness
- Each entry links to relevant product/resource
- Featured snippet optimization (summary list at top)
- Table of contents for long lists

TONE: Enthusiastic, authoritative, engaging
LANGUAGE: Russian
FORMAT: Highly scannable, visual

Make it:
- Easy to skim
- Specific and actionable
- Backed by reasons/data
- Visually appealing with images
```

### Keyword Expansion Prompt
```
Generate additional keyword variations for [YOUR-DOMAIN]:

SEED KEYWORD: [insert keyword]
LANGUAGE: Russian
MARKET: Russia (Yandex focus)

Generate:
1. 5 long-tail variations (3-5 words)
2. 5 question-based keywords (кто, что, где, когда, как, почему)
3. 5 buying intent keywords (купить, заказать, цена, etc.)
4. 5 comparison keywords (vs, лучше, разница, etc.)
5. Related topics and semantically related terms

For each keyword provide:
- Estimated search intent (informational/commercial/transactional)
- Suggested content type (guide/product page/blog)
- Competition estimate (low/medium/high)
```

## Notes for Using These Prompts

### Best Practices:
1. Always customize the prompt with specific details before using
2. Review and edit AI-generated content for accuracy and Russian language quality
3. Ensure all product information is current and correct
4. Run through SEO checklist before publishing
5. Save generated content in `/content/drafts/` first

### Required Information Before Using Prompts:
- Target keyword (from keyword research)
- Target audience definition
- Specific products to mention (if applicable)
- Word count target
- Publication timeline

### After Content Generation:
1. Review for Russian language quality
2. Fact-check all claims and data
3. Optimize images
4. Add schema markup
5. Build internal links
6. Test on mobile
7. Run through pre-publishing checklist
